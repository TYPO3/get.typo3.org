Release Notes for TYPO3 CMS 8.7.6

This document contains information about TYPO3 CMS 8.7.6 which was released on September 7th, 2017.

News

This version is a regression fix release for TYPO3 CMS 8.7.5 LTS concerning the Fluid templating engine and the usage of CKEditor in the backend. This version is a maintenance release and contains bug fixes only.

Upgrading

The usual upgrading procedure applies. No database updates are necessary.
It might be required to clear all caches; the “important actions” section in the TYPO3 Install Tool offers the accordant possibility to do so.
