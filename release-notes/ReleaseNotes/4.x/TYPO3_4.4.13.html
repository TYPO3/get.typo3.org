Release Notes for TYPO3 4.4.13

This document contains information about TYPO3 version 4.4.13 which was released on January 24th 2012.

News

This version is a maintenance release and contains bug fixes only.

Upgrading

The usual upgrading procedure applies. No database updates are necessary.

Changes

Here is a list of what was fixed since 4.4.12:

  • [RELEASE] Release of TYPO3 4.4.13
  • [BUGFIX] md5.js fails with non-ascii-characters (#22328)
  • [BUGFIX] alt_doc.php uses deleted alternative page languages for translations (#31379)
  • [BUGFIX] Add SQL-comparator <> to SQL parser (#32626)
  • [BUGFIX] Invalid query part on menu rendering (#31622)
  • [BUGFIX] @charset must be lowercase in CSS (#32163)
  • [BUGFIX] Random miscalculations in ImageTTFBBox (#21054)
  • [BUGFIX] Localization: Editing in TCEforms shows unstyled original value (#28012)
  • [BUGFIX] Use trimExplode() for pageOverlayFields (#28916)
  • [BUGFIX][SECURITY] XSS vulnerability in BE-User Admin module (#32040)
