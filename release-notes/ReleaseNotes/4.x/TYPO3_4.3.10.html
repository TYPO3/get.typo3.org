Release Notes for TYPO3 4.3.10

This document contains information about TYPO3 version 4.3.10 which was released on December 28, 2010.

News

This version is a maintenance release and contains bugfixes only.

Notes

Since the latest security releases contained (non-security related) regressions and some features were not usable anymore, this new release aims to be a stable package again.
