Release Notes for TYPO3 4.6.17

This document contains information about TYPO3 version 4.6.17 which was released on March 6th 2013.

News

This release is a combined bug fix and security release.

Notes

Due to security issues found in the TYPO3 Core, there was a combined release of TYPO3 4.5.24, 4.6.17, 4.7.9 and 6.0.3.
Find more details in the security bulletin: https://typo3.org/teams/security/security-bulletins/typo3-core/typo3-core-sa-2013-001/
