Release Notes for TYPO3 4.1.12

This document contains information about TYPO3 version 4.1.12 which was released on July 3, 2009.

News

This release is a bugfix release.

Notes

Due to upload problems at SourceForge, the dummy-4.1.12.tar.gz file is missing in this release. However, you can just pick any other dummy archive like e.g. dummy-4.0.12.tar.gz which is identical with this file.

Due to a bug which was reported to us short after the release of TYPO3 versions 4.1.11 and 4.2.7, we stopped the release of the announcement and prepared new versions that fix this (minor) issue. TYPO3 4.0.13 which was already released yesterday was not affected by this bug.
