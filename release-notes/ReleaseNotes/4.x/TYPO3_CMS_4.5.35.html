Release Notes for TYPO3 CMS 4.5.35

This document contains information about TYPO3 CMS 4.5.35 which was released on July 8th, 2014.

News

This release is a bug fix release.

Notes

This is part of a combined release of TYPO3 CMS 4.5.35, 6.1.10 and 6.2.4.
