Release Notes for TYPO3 4.6.15

This document contains information about TYPO3 version 4.6.15 which was released on November 13th 2012.

News

This release is a bug fix release only.

Notes

There was a combined release of TYPO3 4.5.22, 4.6.15 and 4.7.7. These versions include important fixes to regressions introduced within the security releases last week.
