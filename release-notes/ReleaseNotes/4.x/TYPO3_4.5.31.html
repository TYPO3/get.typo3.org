Release Notes for TYPO3 CMS 4.5.31

This document contains information about TYPO3 CMS 4.5.31 which was released on November 26th, 2013.

News

This release is a bug fix release.

Notes

This is part of a combined release of TYPO3 CMS 4.5.31, 4.7.16, 6.0.11 and 6.1.6.
