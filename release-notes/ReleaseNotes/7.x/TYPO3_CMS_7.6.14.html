Upgrading

The usual upgrading procedure applies.
There were changes in DB tables index_phash and sys_file_reference compared to 7.6.11. Log into TYPO3 Install Tool, click on “Compare current database with specification” and apply changes.
It might be required to clear all caches; the “important actions” section in the TYPO3 Install Tool offers the accordant possibility to do so.
