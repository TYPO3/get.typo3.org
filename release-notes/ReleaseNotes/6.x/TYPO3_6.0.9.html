Release Notes for TYPO3 6.0.9

This document contains information about TYPO3 CMS 6.0.9 which was released on September 4th 2013.

News

This release is a combined bug fix and security release.

Notes

Due to security issues found in the TYPO3 Core, there was a combined release of TYPO3 CMS 6.0.9 and 6.1.4.
Find more details in the security bulletin: https://typo3.org/teams/security/security-bulletins/typo3-core/typo3-core-sa-2013-003/
